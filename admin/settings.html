MailerLite CRM & Newsletter

Newsletter signups, customer accounts, guest checkouts, and Barefoot Remembers entries can sync to MailerLite. Add the API key and group variables in your Vercel project settings for secure server-side syncing.

Checking...

Environment VariablePurpose
MAILERLITE_API_KEYRequired API token from MailerLite
MAILERLITE_GROUP_ALL_CONTACTSOptional master group for every synced contact
MAILERLITE_GROUP_SUBSCRIBERSOptional group for newsletter and popup signups
MAILERLITE_GROUP_CUSTOMERSOptional group for account signups and order customers
MAILERLITE_GROUP_REMEMBERSOptional group for Barefoot Remembers registrations

Group values can be MailerLite group IDs or exact group names. If you provide a group name that does not exist yet, the site will create it automatically.

API & Integrations

This dashboard is backed by a REST API at /api/*, secured with a JWT bearer token. Use these endpoints to connect other tools (e.g. inventory software, a mobile app):

EndpointPurpose
POST /api/auth/loginAuthenticate, returns JWT
GET/POST /api/productsList / create products
GET/PUT/DELETE /api/products/:idManage a single product + images
GET/POST /api/mediaMedia library (Vercel Blob storage)
GET/POST /api/ordersOrder history / create order
PATCH /api/orders/:idUpdate order status
GET/POST /api/commentsReviews & blog comments
POST /api/subscribeNewsletter and popup email signup
POST /api/paystack/initializeStart a Paystack transaction
GET /api/paystack/verifyVerify payment after redirect
POST /api/paystack/webhookPaystack server-to-server webhook
